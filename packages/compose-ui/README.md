
# @lglab/compose-ui


## Install

```
pnpm add @lglab/compose-ui
```

or

```
npm i @lglab/compose-ui
```

or

```
yarn add @lglab/compose-ui
```

or

```
bun add @lglab/compose-ui
```

## Quick Start

```
import { Button } from '@lglab/compose-ui'

const Page = () => {
  return (
    <Button>Click</Button>
  );
}
```
