## Overview

Compose UI follows the [llms.txt standard](https://llmstxt.org/) to make our documentation easy for AI tools to consume. Whether you're using Cursor, Claude, ChatGPT, or another AI assistant, you can give it full context about our components.

## Available Files

**[llms.txt](/llms.txt)**  
Index with links to all component and block docs. Use for quick reference and agent navigation.

**[llms-full.txt](/llms-full.txt)**  
Complete documentation in one file. Paste into ChatGPT/Claude or add to Cursor @Docs.

**[/components/\[component\].md](/llms/button.md)**  
Individual component markdown files. Use for focused context on specific components.

**[/blocks/\[block\].md](/llms/blocks/statistics.md)**  
Individual block markdown files. Use for focused context on specific block patterns.

## Usage

### Cursor

Add Compose UI to your Cursor docs for autocomplete and chat context:

1. Open Cursor settings
2. Go to **Features â†’ Docs**
3. Click **Add new doc**
4. Enter: https://compose-ui.dev/llms-full.txt

Now you can reference it with `@Docs` in Cursor chat.

### Claude / ChatGPT

Copy the contents of [/llms-full.txt](/llms-full.txt) and paste it into your conversation, or share the URL directly if the tool supports fetching.

### Claude Code / Windsurf

Point the agent to our llms.txt:

```
Read https://compose-ui.dev/llms.txt for Compose UI documentation
```

### Per-Component Context

Component pages include a **View Markdown** link and **Copy Markdown** button in the header. Use these to grab focused documentation for a specific component without loading the entire library docs.

You can also access markdown directly with **.md** URLs, for example:

- **https://compose-ui.dev/components/button.md**
- **https://compose-ui.dev/blocks/statistics.md**

## Tips for Best Results

- **Be specific**: Tell the AI you're using Compose UI, not Base UI directly
- **Include examples**: Our markdown files include full code examples for each variant
- **Advanced customization**: For deeper API questions, component markdown files link to underlying API documentation

## Example Prompts

```
Using Compose UI, create a Dialog with a form that has email and password fields.
```

```
Show me how to use Compose UI Tabs with icons and vertical orientation.
```
